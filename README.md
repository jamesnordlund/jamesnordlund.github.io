# Curriculum Vitae Website

A professional, Emacs-inspired CV website with a retro coding aesthetic.

## Features

- **Emacs-inspired design**: Dark theme with classic terminal colors
- **Professional presentation**: Clean layout reflecting senior executive level
- **Retro coding vibe**: Terminal-style interface with monospace fonts
- **Responsive**: Works on desktop, tablet, and mobile devices
- **Print-friendly**: Optimized CSS for printing

## Customization

### Personal Information

Edit `index.html` to update:
- Name (line with `JAMES [SURNAME]`)
- Contact information (email, LinkedIn, GitHub)
- Professional experience
- Education
- Skills and achievements

### Styling

Modify `styles.css` to adjust:
- Colors (defined in `:root` variables)
- Fonts
- Spacing and layout
- Terminal appearance

## Local Development

Simply open `index.html` in a web browser. No build process required.

For a local server (optional):
```bash
python -m http.server 8000
# or
npx serve
```

Then visit `http://localhost:8000`

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
